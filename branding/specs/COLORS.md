## 3. Deployment Instructions

### Windows (Microsoft 365)
1. Navigate to **Design** > **Variants** (Dropdown) > **Colors** > **Customize Colors...**.
2. Assign the **Primary (Base)** RGB values to the **Accent 1 through Accent 6** slots.
3. Set "Hyperlink" to Woodfine Blue (22, 70, 121) and "Followed Hyperlink" to Woodfine Purple (124, 70, 140).
4. Save the theme as **"Woodfine Corporate"**.

### macOS (Microsoft 365)
1. Navigate to **Design** > **Colors** > **Customize Colors...**.
2. Click each Accent color box. In the macOS Color Picker, select the **RGB Sliders** icon.
3. Enter the **Primary (Base)** RGB values manually.
4. Save the theme as **"Woodfine Corporate"**.
